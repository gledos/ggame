# 衣服品牌问题

观众提醒直播中的 PDD，说明衣服品牌的污点问题，PDD 迅速更换了衣服，不过观众还是挑出了毛病，就这样换了两次衣服。不过观众依然指出这个服装品牌有污点，PDD 觉得是不是观众在开玩笑，于是打开搜索引擎网页，搜索相关新闻，确认了有这样的新闻，于是更换了一个没有外露商标的衣服，并自证这不是 H&M 或者优衣库的衣服。
