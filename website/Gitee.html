    Gitee

    简介#

    Gitee码云 是基于 Git 的代码托管和协作开发平台，由开源中国于 2013 年推出，主要模仿的是 GitHub。1

    强制私有 codes#

    Gitee 曾有个 codes 功能，类似于 GitHub Gist，能创建代码片段并公开分享，URL 大概是 https://gitee.com/example_name/codes/example_id/raw/。2

    2022 年之前的某个未知时间，Gitee 将所有的 codes 代码强制非公开化了，导致一些安装脚本不能使用。3

    仓库公开须知#

    2022年1月17日 之前的未知时间，4 Gitee 添加了「仓库公开须知」的协议，需要同时勾选三个协议，才能将仓库设置为公开仓库。详细内容如下：5

    仓库公开须知
    • 我承诺此仓库内容不违反任何国家法律法规
    • 我承诺此仓库内容不侵犯任何单位及个人的版权和权益
    • 我承诺本人拥有该仓库所有内容的版权；如仓库内容引用他人作品，我承诺所引用内容均确认为合法内容, 不存在版权问题，且对引用内容做了详细标注说明

    如违反以上承诺，Gitee保留追究相关责任的权利。更多信息请访问《Gitee.com网站使用条款》

    2022年5月18日，需要签署「仓库公开须知」协议，并等待审核，仓库才能公开。如果没有审核，仓库就会强制处于私有状态，访客查看会显示「为了营造绿色健康的开源社区，推动中国开源生态建设，当前仓库仅限成员访问。」6

    禁止图床#

    2022年3月25日，Gitee 对仓库中的图片添加了禁止其他网页查看仓库中的图片的机制，7891011 并且有人发现了网站发生了变化，从命令行访问会产生一个过渡页面，需要跳转一下，这导致命令行安装软件脚本不再正常工作。而且导入 Github 仓库到 Gitee 时也不能直接选择「公开」仓库， 需要在设置里同意「仓库公开须知」才能公开。12

    在 Gitee 的《网站使用条款》中，确实明确的写出了「禁止盗链 奥思网络禁止擅自利用本网站内容的网页构架系统的行为。奥思网络保留断开任何未经授权指向本网站的超文本链接或网页构架系统的权利。」13

    「AV」审查#

    2022年5月19日，知乎用户「王德福」讲了个笑话：14

    王德福的知乎回答

    本次提交代码中：

    // user.vue line:87

const saveData = () => {

    因包含敏感词 av，本次PR被拒绝，建议修改后再尝试提交。

    感谢您的理解。

    Gitee 审核团队

    随后被不规范转载，导致一些人误以为这是真实的系统消息。15

    随后在 2022年6月2日，Gitee 背后的开源中国公司开设了直播，节目中指出了这个消息是虚假的，并称：「发布这个消息的作者也已经跟我们道歉了，这纯粹就是他个人因为好玩发的调侃。」16

    不过从 2022年6月 开始，有人发现 Gitee Pages 会审查词语：17

    原因是关键词审查（敏感词），至少会审查这些内容「save」「以下注意」「在手机上访问」，这些文字的加粗部分就是关键词。1819

    Gitee Pages 关闭#

    最早在 2024年4月29日，有人发现 Gitee Pages 功能下线，相关按钮变成灰色，无法开启仓库的 Gitee Pages 功能。21

    稍后有人向客服询问：

    • 2024年4月30日 客服说：「不着急」。21
    • 2024年5月1日 客服说：「下线了」。21
    • 2024年5月6日 客服说：Pages 服务短期内大概率无法上线了。22
